# Metric/Imperial Unit Converter

A fast, responsive, and user-friendly web application built to convert values between the Metric and Imperial measurement systems in real time. 

🌐 **Live Website:** (https://voidaquacore-web.github.io/unit-converter/)

## ✨ Features
* **Length:** Converts between meters and Feet.
* **Mass/Weight:** Converts between Kilograms and Pounds.
* **Volume:** Converts between Liters and Gallons.
* **Instant Calculation:** Automatic, real-time conversion as you type.
* **Responsive Layout:** Works flawlessly across mobile, tablet, and desktop devices.

## 🛠️ Technologies Used
* **HTML5:** Core semantic structure.
* **CSS3:** Custom styling and adaptive grid layout.
* **JavaScript (ES6):** Dynamic logic and instant conversion algorithms.

## ⚙️ Core Logic Examples
The application uses standard math formulas for high-accuracy conversions, including:
* **Meters to Feet:** $\text{value} \times 3.28084$
* **Kilograms to Pounds:** $\text{value} \times 2.20462$
